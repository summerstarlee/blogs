# npm 相关

  1. 更新最新的正式版本
npm install npm@latest -g
1
  1. 更新到最新的体验版本
npm install npm@next -g
1
  1. 查看全局安装的模块
npm list -g --depth 0
1

# yarn global 命令

  1. global 必须跟在 yarn 后面
  2. 全局添加模块
yarn global add xxx
1
  1. 列出全局安装模块
yarn global list
1
  1. 全局移除某个模块
yarn global remove xxx
1
